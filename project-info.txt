Project Name

AutoJokeX вЂ“ Autonomous Twitter (X) Joke Posting Bot

1. Project Overview
Objective

Build a fully automated bot that posts one funny joke on X (Twitter) every hour without any human intervention.

The bot should:

Run 24Г—7.
Automatically publish jokes every hour.
Never post the same joke twice until all jokes are exhausted.
Maintain complete logs.
Recover automatically from crashes.
Be production-ready.
Use no paid AI APIs.
Use a local joke database instead of AI-generated content.

The system should be designed in a modular way so future upgrades (AI joke generation, image memes, multiple accounts, analytics, etc.) can be added without major code changes.
